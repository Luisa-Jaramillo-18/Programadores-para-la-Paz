Prueba Ruta POST

Respuesta: Al enviar una petición POST a la ruta 
http://localhost:3000/registro utilizando REST Client, 
el servidor recibió correctamente los datos enviados 
en formato JSON. En la prueba envié el nombre "Luisa" 
y el mensaje "Hola comunidad". Como respuesta, el servidor
devolvió el siguiente objeto JSON:

{
  "estado": "Datos recibidos",
  "nombre": "Luisa",
  "mensaje": "Hola comunidad"
}

Esta respuesta indica que el servidor recibió correctamente 
la información enviada y la devolvió en formato JSON. De esta
manera, se comprobó que la ruta POST funciona correctamente y 
que puede recibir y procesar datos enviados por el cliente.
