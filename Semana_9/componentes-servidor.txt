Funciones de Componentes

Sistema operativo:
Es el software que administra los recursos del computador y permite ejecutar programas. 
También facilita la comunicación entre el hardware y las aplicaciones.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. 
Se utiliza para crear y ejecutar aplicaciones del lado del servidor.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas, procesa las solicitudes 
de los usuarios y envía las respuestas correspondientes.

Puerto:
Es el punto de comunicación que utiliza el servidor para recibir solicitudes. En esta 
práctica se utilizó el puerto **3000** para acceder a la aplicación desde el navegador.

Logs:
Son los mensajes que el servidor muestra en la terminal durante su ejecución. Ayudan a 
verificar que el servidor funciona correctamente y a identificar posibles errores.
